# Davčni kalkulator za Freelancerje

**[Odpri kalkulator](https://knagode.github.io/freelancers_and_taxes/)**

Interaktivna aplikacija za primerjavo obdavčitve freelancerjev v 18 državah. Vnesi letni ali mesečni prihodek in takoj vidiš koliko ti ostane po plačilu davkov in prispevkov v vsaki državi.

## Funkcionalnosti

- Preklop med mesečnim in letnim prikazom
- Slider 0 - 160.000 EUR
- Podroben izračun za vsako državo (prispevki, dohodnina, neto)
- Vizualni prikaz razdelitve prihodkov
- Primerjava vseh držav z razvrstitvijo po neto znesku
- Prednosti in slabosti vsake države
- Priporočen tip podjetja za freelancerja

## Opozorilo

Podatki so okvirni in ne nadomeščajo uradnega davčnega svetovanja. Primerjave temeljijo na poenostavljenih modelih za samozaposlenega z nizkimi dejanskimi stroški.
